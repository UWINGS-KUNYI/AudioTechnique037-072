# 音響技術 Audio Technique 雜誌掃描整理（第 37～72 期）

本專案整理自一本創刊於 1976 年元月的中文音響技術雜誌《音響技術》（Audio Technique）。這些資料為網路上廣泛流傳的掃描檔（JPG 格式），特此分期歸檔，以利保存、研究與非營利分享使用。

> 🔸 本倉庫收錄第 37 期（1979 年 1 月）至第 72 期 （1981 年 12 月）的雜誌掃描圖像。
